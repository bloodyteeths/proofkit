Badge Data Structure

Each badge object should contain:

  • src (required): Path to the badge SVG image
  • alt (required): Accessible description of the badge
  • href (optional): External link for more information

Features

  • Mobile-friendly horizontal scrolling
  • Hover and focus effects with shadows
  • External links open in new tabs
  • Accessible with proper ARIA labels
  • Responsive design (96×64px on desktop, 80×54px on mobile)
  • Support for reduced motion preferences
  • High contrast mode compatibility
